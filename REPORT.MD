# Delivery Performance Analysis – Streamline Logistics Solutions

## Author
Modinat Omolara Buhari

## Introduction
Delivery performance is a critical factor in logistics and supply chain management. Companies rely on timely and efficient deliveries to maintain customer satisfaction and operational efficiency.

This project focuses on analyzing delivery performance data to understand delays, identify inefficiencies, and improve logistics operations.

## Background
In today’s competitive logistics industry, delivery delays can lead to customer dissatisfaction, increased costs, and loss of business reputation.

Analyzing delivery performance helps organizations:
- Monitor operational efficiency
- Identify problem areas
- Improve decision-making using data

This project simulates how a logistics company can use Excel to track and improve delivery performance.

## Project Overview
This project analyzes delivery performance for Streamline Logistics Solutions using Excel.

The goal is to identify inefficiencies, delays, and trends in delivery operations and provide actionable insights to improve performance.

## Objectives
- Analyze delivery times and delays
- Identify key performance indicators (KPIs)
- Evaluate on-time delivery performance
- Provide recommendations for improvement

## Dataset
The dataset includes:
- Order ID
- Delivery Date
- Expected Delivery Date
- Delivery Status
- Region
- Customer information

## Key Performance Indicators (KPIs)
- On-Time Delivery Rate
- Late Deliveries
- Average Delivery Time
- Regional Performance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Dashboards

## Dashboard & Analysis
An interactive Excel dashboard was created to visualize:
- Delivery trends over time
- Delay patterns
- Performance by region
- KPI summaries

## Key Insights
- A significant number of deliveries were delayed beyond expected dates
- Certain regions experienced more delays than others
- Peak delivery periods showed higher inefficiencies

##  Key Insights

- Average delivery time is **152 minutes**
- Average delays recorded at **15 per delivery cycle**
- Customer feedback distribution:
  - Positive: **35%**
  - Neutral: **34%**
  - Negative: **31%**
- Route1 has the highest delivery time (**161 minutes**)
- Expedited allocation rules resulted in higher delays (**154 minutes**) compared to standard methods

## Recommendations
- Improve route planning for high-delay regions
- Optimize delivery scheduling during peak periods
- Monitor KPIs regularly using dashboards
